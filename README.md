# What is this project about?
This is a simple minesweeper application written in Python with a GUI made using [PyQt5](https://pypi.org/project/PyQt5/).

# Features
Features include:
- Playing the game with randomized mine placement
- Resetting the game to start from the beginning
- Keeping track of win and lose conditions
- Being able to place flags on the board (right-click)
- Amount of turns made is tracked and presented to the player
- Automatic solving of the board
